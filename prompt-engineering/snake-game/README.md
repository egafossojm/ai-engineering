# Snake Game

### Prepare Environment
- Create a virtual pyhton environment
```bash
$ python3 -m venv venv
```
- Activate the environment
```bash
$ source venv/bin/activate
```
- Install pygame library
```bash
$ pip install pygame
```

### Run the game
```bash
$ python3 snake_game.py
```

### Game controls
- SPACE key → start
- Arrow keys → move
- R key → restart
- ESC → quit
